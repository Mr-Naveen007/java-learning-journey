\# 📅 Day 29 – Java Serialization \& Deserialization



\## 🚀 Objective



Implement Java object serialization and deserialization using file streams.



\---



\## 📁 Project Structure



```

day29-serialization/

├── src/

│   ├── User.java

│   ├── SerializeDemo.java

│   └── DeserializeDemo.java

├── user.ser

└── README.md

```



\---



\## ▶️ How to Run



1\. Compile:



```

javac src/\*.java

```



2\. Run Serialization:



```

java src.SerializeDemo

```



3\. Run Deserialization:



```

java src.DeserializeDemo

```



\---



\## 🧪 Sample Output



```

Object serialized successfully

Name: Naveen

Age: 22

```



\---



\## 📌 Key Learning



\* Object → byte stream (serialization)

\* Byte stream → object (deserialization)

\* Foundation for backend data transfer
