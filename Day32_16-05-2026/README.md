\# JSON to Java Object Mapping using Jackson



This project demonstrates how to:



\- Call a real REST API using Java HttpClient

\- Deserialize JSON into Java objects using Jackson

\- Serialize Java objects back into JSON



\## Technologies Used



\- Java 17

\- Maven

\- Jackson Databind



\## API Used



https://api.github.com/users/octocat



\## Features



\- JSON → Java Object Mapping

\- Java Object → JSON Conversion

\- Real GitHub API Integration

\- Unknown JSON Property Handling
