\# Day 26 - Exception Handling (ATM Simulation)



\## 📌 Objective

Learn how to handle runtime errors using try-catch and prevent program crashes.



\## ⚙️ Features

\- Withdraw money

\- Deposit money

\- Balance tracking

\- Exception handling for:

&#x20; - Insufficient balance

&#x20; - Invalid amount (<= 0)



\## 🧠 Concepts Learned

\- try block (risky code)

\- catch block (error handling)

\- throw keyword (manual exception creation)

\- IllegalArgumentException

\- Control flow during exceptions



\## 💻 Example Scenarios



\### Valid Withdraw

Withdraw successful. Remaining balance: 500.0



\### Invalid Withdraw

Error: Insufficient balance



\### Invalid Input

Error: Amount must be positive



\## 🚀 How to Run

1\. Open project in IntelliJ

2\. Run ATM.java
